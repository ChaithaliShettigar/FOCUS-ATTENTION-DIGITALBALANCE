# Group & Search Features - Quick Start Guide

## What's New?

### 1. **Unique Usernames** 
Every user now has a unique username (like @johndoe) that they set during registration.

### 2. **Group Codes**
Every group now has an auto-generated 8-character code (like ABC12DEF). Share this code with others to let them join your group!

### 3. **Real-time Username Search**
Type in the search bar and see user suggestions appear in real-time as you type!

---

## How to Register

1. Go to the registration page
2. Fill in:
   - **Name**: Your full name
   - **Username**: Your unique username (3-20 characters, letters/numbers/hyphens/underscores)
   - **Email**: Your email address
   - **Password**: Strong password (8+ chars, uppercase, number, special char)
3. Click Register

### Username Rules:
- 3-20 characters long
- Can include: letters, numbers, hyphens (-), underscores (_)
- Cannot include: spaces or special characters (except - and _)
- Examples: `john_doe`, `JohnDoe123`, `john-doe`

---

## How to Create a Group

1. Click **"Create Group"** button
2. Enter group name (e.g., "CSE101 Study Group")
3. Click **Create**
4. Your new group appears with a **Group Code**
5. Copy the code and share with others

### Group Code
- 8 characters (uppercase letters and numbers)
- Example: `ABC12DEF`
- Share this with friends to let them join
- Click the copy icon to copy to clipboard

---

## How to Join a Group

1. Click **"Join Group"** button
2. Paste or type the group code (8 characters)
3. Click **Join**
4. You're now a member!

### Tips:
- Get the code from a friend who created the group
- Codes are case-insensitive (ABC12DEF = abc12def)
- If code doesn't work, ask the group creator for the correct code

---

## How to Search for Users

### Method 1: Real-time Suggestions
1. Go to **Search** page
2. Start typing in the search bar (name, email, or username)
3. See suggestions appear as you type
4. Click a suggestion to view that user's profile

### Method 2: Full Search
1. Type your search query completely
2. Click **Search** button
3. See all matching results
4. Click a result to view their profile

### Search Tips:
- Search by **name**: "John" or "Jane"
- Search by **username**: "@johndoe" or just "johndoe"
- Search by **email**: "john@example.com"
- Results show users with public focus scores

## Manage Your Groups

### View Group Members
1. Click on a group card
2. See all members with their:
   - Name and username
   - Focus score
3. Copy group code to share

### Leave a Group
1. Open group details
2. Click **Leave Group** button
3. Confirm

### Delete a Group (Creator Only)
1. Open group details
2. Click **Delete** button (only shows if you created the group)
3. Confirm (cannot be undone!)

## Common Questions

### Q: Can I change my username?
A: Not yet, but you can contact support if needed.

### Q: Can multiple users have the same name?
A: Yes, but each user must have a unique **username**.

### Q: What if I forget the group code?
A: Open the group details - the code is displayed there. You can copy it anytime.

### Q: Can I see my group code?
A: Yes! Click on any group you created or joined to see the code.

### Q: What if the group code doesn't work?
A: Make sure you typed it correctly (it's case-insensitive but must be exactly 8 characters).

### Q: Can I search for private users?
A: No, search results only show users with public focus scores enabled. They can enable this in their Profile settings.

### Q: Can I change a group code?
A: Not yet, but each group has a unique code that's permanent.
